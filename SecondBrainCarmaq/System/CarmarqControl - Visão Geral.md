# CarmarqControl

## Resumo

O **CarmarqControl** é um sistema de gestão operacional com foco em organização de processos, rastreabilidade e suporte a múltiplos perfis de uso.

## Stack principal

- React
- Spring Boot
- PostgreSQL

## Papéis do sistema

- `Owner`
- `Technician`
- `Financial`

## Direção do sistema

Este projeto deve manter alinhamento entre:

- contexto de negócio;
- regras de acesso;
- implementação técnica;
- documentação persistente.

## Organização no vault

- trabalho diário em `01 - Daily/`
- features em `02 - Projects/Features/`
- responsabilidade contínua em `03 - Areas/Desenvolvimento/`
- conhecimento durável em `04 - Knowledge/`
- agentes e automação em `AI/`
- regras centrais em `Rules/`
- documentação base em `System/`
